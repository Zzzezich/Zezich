DNS (Domain Name System) - это компьютерная распределенная система для получения информации о доменах,
информатизации о маршрутизации почтыи обслуживающих узлах для протоколов в домене.

HTTP - протокол высшего уровня сетевой модели ISO (модель, посредством который, различные сетевые устройства 
могут взаимодействовать друг с другом) для передачи данных.

Существуют следующие версии HTTP:
- 0.9 – самая ранняя, работавшая в начале 90х
- 1.0 – основа современной версии
- 1.1 – основная версия, принятая с 1999, позволяет посылать несколько запросов за одно соединение, более просто организована система хостинга
- 2.0 – с 2015, в настоящее время около 50% сайтов поддерживают его, бинарный, позволяет серверу высылать содержимое, которе ещё не было запрошено
- 3.0 – протокол будущего, поддерживается некоторыми сайтами и браузерами
Все версии обладают обратной совместимостью

HTTPS - расширение протокола HTTP для поддержки шифрования в целях повышения безопасности.

Их различие в том, что HTTPS - это только расширение протокола HTTP, а не отдельный протокол. Еще, HTTP использует
порт TCP-80, а HTTPS порт TCP-443.
